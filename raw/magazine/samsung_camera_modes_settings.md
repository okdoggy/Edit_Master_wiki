# Samsung Support — Camera modes and settings — 상황별 촬영/보정 팁

이 파일은 더 이상 단순 소스 메타데이터가 아니라, 실제 촬영/보정 때 바로 써먹는 레시피 형태로 정리한다.

## 바로 쓰는 팁 카드

### Tip 1. 달 촬영 — 사용자가 예시로 든 “10배 줌 + Pro mode” 케이스

**촬영/작업 순서**
- 카메라 → More → Pro 또는 Expert RAW 진입
- 10x 망원으로 달을 화면에 크게 배치
- 삼각대/난간에 고정 후 MF를 무한대 부근에 맞춤
- 달 표면이 회색 질감으로 보일 때까지 셔터를 빠르게 조정

**추천 시작값 / 조작값**
- ISO 50~100 시작
- Shutter 1/125~1/500s 시작
- Focus MF 무한대
- WB 4000~5200K 또는 Auto
- Timer 2초

**보정 루틴**
- Highlights -50~-100
- Texture/Clarity +10~+30
- 필요 시 흑백 변환으로 색수차 감소

**주의할 점**
- Night mode로 달을 밝게 만들면 표면이 날아갈 수 있음
- 디지털 100x 결과는 기록용/재미용과 작품용을 구분

**확실성:** 숫자는 달 촬영 실전 시작점; Pro mode 수동 제어는 Samsung 공식 지원.
**근거:** Samsung Pro mode 문서는 ISO/셔터/노출/WB 수동 제어를 공식 설명한다.

### Tip 2. 갤럭시로 달 표면을 선명하게 찍고 싶을 때

**촬영/작업 순서**
- 삼각대 또는 고정대를 쓰고, 기본 카메라의 Pro mode/Expert RAW로 들어간다.
- 가능하면 10x 광학 망원부터 시작하고, 디지털 확대는 결과 확인 후 최소한만 쓴다.
- 초점은 수동으로 무한대 쪽에 두고, 달 표면이 하얗게 날아가지 않을 때까지 셔터속도를 빠르게 조정한다.
- 한 장만 믿지 말고 10~20장을 찍어 대기 흔들림이 적은 컷을 고른다.

**추천 시작값 / 조작값**
- ISO: 50~100 시작, 너무 어두우면 200까지
- Shutter: 1/125~1/500s 시작
- Focus: MF 무한대 부근
- WB: 4000~5200K 시작 또는 Auto 후 RAW 보정
- Zoom: 10x 광학 우선

**보정 루틴**
- Lightroom에서 Highlights -50~-100, Whites -10~-40로 표면 디테일 복구
- Texture/Clarity +10~+30, Sharpening은 과하면 테두리 halo 발생
- 필요하면 흑백 전환으로 색수차를 줄인다.

**주의할 점**
- Night mode로 달을 밝게 만들면 표면이 하얗게 뭉개질 수 있다.
- 100x 스페이스줌 결과는 AI/처리 개입이 크므로 기록용과 작품용을 구분한다.

**확실성:** 수치값은 NASA/일반 달 노출 원칙과 스마트폰 Pro mode 실전 시작점; Samsung 공식은 Pro/Expert RAW 수동 제어 지원.
**근거:** Samsung Pro mode는 ISO/셔터/노출/WB 수동 제어를 제공하고, 달은 밝은 피사체라 낮은 ISO와 빠른 셔터가 유리하다.

### Tip 3. 갤럭시로 별·은하수·어두운 밤하늘을 찍을 때

**촬영/작업 순서**
- 도시광을 피하고, 삼각대를 고정한 뒤 Expert RAW의 Astrophoto 또는 Pro mode를 사용한다.
- 초점은 수동으로 별이 가장 작고 또렷해지는 지점에 맞춘다.
- 셔터를 길게 열고, ISO는 하늘이 희뿌옇게 뜨지 않는 선에서 올린다.
- 촬영 후 100% 확대해 별이 선이 되는지 확인하고 셔터를 줄인다.

**추천 시작값 / 조작값**
- Shutter: 15~30s 시작
- ISO: 800~1600 시작, 밝은 지역은 400~800
- Focus: MF 무한대 부근
- WB: 3200~4200K 시작
- Timer: 2~5초

**보정 루틴**
- Exposure는 소폭, Contrast/Dehaze +5~+20
- Color Noise Reduction을 먼저 확인
- 하늘은 Mask로 분리해 너무 파랗게 만들지 않는다.

**주의할 점**
- 손持ち 장노출 금지
- ISO를 무조건 높이면 노이즈와 보라색/초록색 얼룩이 늘어난다.

**확실성:** Samsung Newsroom/Community 계열의 Expert RAW·장노출 절차 + 실전 시작점.
**근거:** Samsung Newsroom은 Expert RAW에서 15초 셔터 예시를, Samsung 자료는 Pro/Expert RAW 수동 제어를 제시한다.

### Tip 4. 인물 주변에 네온/라이트 페인팅 효과를 만들 때

**촬영/작업 순서**
- 삼각대에 고정하고 피사체에게 촬영 중 절대 움직이지 말라고 안내한다.
- Expert RAW 또는 Pro mode에서 수동 초점으로 피사체에 맞춘다.
- 작은 LED/튜브 라이트를 들고 피사체 뒤나 옆을 천천히 지나가며 빛 궤적을 만든다.
- 배경이 너무 밝으면 ISO를 낮추고, 빛 궤적이 약하면 ISO나 라이트 밝기를 올린다.

**추천 시작값 / 조작값**
- Shutter: 15s — Samsung Newsroom 예시
- ISO: 라이트 밝기에 따라 50~400 시작
- Focus: Manual
- Tripod: 필수
- Timer: 2초

**보정 루틴**
- Highlights를 낮춰 라이트 중심부 날림을 줄인다.
- Subject mask로 얼굴만 노출/화이트밸런스 보정
- 배경은 Dehaze/Contrast를 약하게 올려 빛 궤적을 살린다.

**주의할 점**
- 피사체가 움직이면 얼굴이 흐려진다.
- 라이트가 카메라를 직접 향하면 큰 플레어가 생긴다.

**확실성:** source-supported: Samsung Newsroom이 15초 셔터와 Expert RAW/light drawing 절차를 제시.
**근거:** Samsung Newsroom Malaysia — Pro Mode & Expert RAW App on Galaxy S23 Ultra.

### Tip 5. 무엇을 찍어도 사진이 산만하고 초점/밝기가 흔들릴 때

**촬영/작업 순서**
- 렌즈를 먼저 닦고, 카메라 설정에서 격자(Grid)와 수평(Level)을 켠다.
- 주 피사체를 화면 교차점 또는 중앙보다 약간 옆에 둔다.
- 피사체를 탭해서 초점과 노출 기준을 정하고, iPhone은 길게 눌러 AE/AF Lock을 건다.
- 하이라이트가 날아가면 화면 노출 슬라이더를 살짝 내려 밝은 부분 디테일을 살린다.

**추천 시작값 / 조작값**
- EV/노출 보정: 밝은 하늘·흰 벽·네온이 있으면 -0.3~-1.0부터 테스트
- 타이머: 삼각대/거치 시 2~3초
- 줌: 가능하면 0.5x/1x/2x/3x/5x 같은 광학 배율만 사용

**보정 루틴**
- 수평/크롭을 먼저 맞춘 뒤 노출·색을 조정한다.
- 밝은 영역 복구가 필요하면 Highlights를 낮추고 Shadows는 과하게 올리지 않는다.

**주의할 점**
- 디지털 줌으로 먼저 당기지 말고, 고해상도 촬영 후 크롭한다.
- 초점 확인 없이 연속 촬영하지 않는다.

**확실성:** source-supported 일반 원칙 + 실전 시작점.
**근거:** Apple의 AE/AF/Night mode 흐름, Samsung Pro mode, TIME/PetaPixel/Digital Camera World류의 기본 팁을 통합.

## 추가 검증 촬영 레시피 — Galaxy

### Galaxy 달 사진: Scene Optimiser + 100x + Zoom Lock

**상황**
- Pro mode 수동값보다, 갤럭시의 AI/슈퍼해상도 달 인식 기능을 활용해 쉽게 달을 크게 찍고 싶을 때.

**촬영/작업 순서**
- Scene Optimiser를 ON으로 둔다.
- 달을 화면 중앙에 두고 100x zoom까지 확대한다.
- 양손으로 가로로 잡거나 지지면에 고정한다.
- Zoom Lock의 노란 테두리가 뜰 때까지 안정시킨 뒤 촬영한다.

**주의점**
- 구름에 달이 가리면 인식이 약해질 수 있다.
- AI 보정/합성적 처리 개입이 싫으면 Scene Optimiser를 OFF로 두고 Pro/Expert RAW 수동 촬영을 사용한다.

### Galaxy 야간 도시 장노출: Michael Shainblum식 시작값

**상황**
- 야간 도시 풍경, 높은 전망대, 도로 빛줄기, 건물 라인을 안정적으로 찍고 싶을 때.

**촬영/작업 순서**
- Pro mode로 전환한다.
- 삼각대 또는 고정면에 올린다.
- Focus를 infinity로 둔다.
- 장면이 너무 밝으면 셔터를 줄이고, 너무 어두우면 ISO를 조금 올린다.

**출처 기반 시작값**
- Shutter: 8s.
- ISO: 50.
- WB: 4300K.
- Focus: infinity.

**주의점**
- 움직이는 사람/차가 많으면 8초는 번짐이 커진다. 번짐을 의도할 때만 사용한다.

### Galaxy 근접/매크로 흐림 줄이기

**상황**
- 접사에서 피사체 주변이 이상하게 흐려질 때.

**촬영/작업 순서**
- S23/Fold5 계열은 0.6x, Flip5 계열은 0.5x부터 시도한다.
- 피사체가 30cm 이내라면 약 2.5cm 더 물러난다.
- 스마트폰을 세로로 유지하고, 피사체와 평행하게 맞춘다.

**주의점**
- 비스듬한 각도는 주변부 흐림을 크게 만든다.

### Galaxy forced perspective / 창의적 착시

**상황**
- 소품과 원근으로 SNS용 착시 사진을 만들 때.

**촬영/작업 순서**
- Wide-angle을 기본으로 쓴다.
- 배경은 단순하게 둔다.
- 피사체를 카메라에 매우 가깝게 또는 멀게 둬 크기 착시를 만든다.
- mirror, prism, slinky 같은 소품을 활용한다.
- 손이 바쁘면 voice shutter 또는 리모컨을 쓴다.

## 모드별 공식 검증값 추가

### Samsung Food mode 세부 조작

**공식 경로**
- Camera → MORE → FOOD.

**촬영 절차**
- 음식이 highlighted area 안에 들어오도록 프레임을 잡는다.
- Thermometer 아이콘으로 색온도/색감을 조정한다.
- Blur effect로 음식 주변이나 배경을 흐린다.

**실전 팁**
- 흰 접시가 노랗게 뜨면 Thermometer를 차갑게 조정한다.
- 여러 접시가 있으면 Blur effect를 약하게 하거나 일반 Photo도 같이 찍는다.

### Samsung Night mode 세부 원리

**공식 경로**
- Camera → More → Night → Capture.

**공식 확인 포인트**
- Samsung Night Mode는 AI multi-frame processing으로 여러 이미지를 합성한다.
- Samsung night-mode guide는 30 images를 하나로 결합하는 설명을 제공한다.
- 안정적인 자세 또는 삼각대가 결과를 크게 좌우한다.

**실전 팁**
- 손持ち: 짧은 Night mode, 사람은 멈추게 하기.
- 삼각대/고정: 더 긴 Night mode 또는 Pro mode 장노출.
- 달/네온처럼 매우 밝은 피사체는 Night mode보다 수동 노출이 유리할 수 있다.

### Samsung Pro mode / Expert RAW 범위

**공식 확인 조작**
- ISO, shutter speed, exposure, color tone, manual focus, white balance.

**공식 예시 범위**
- ISO 50–800.
- Shutter 1/24000–10s.
- White balance 2300K–10000K.
- Manual focus: 가까운 거리부터 far away.

**Expert RAW 예시**
- 별도 다운로드 앱/기능.
- 더 세밀한 제어와 RAW 기반 편집.
- Samsung S25 자료 예시: ISO 800, 1/125s, AF-C, highlights/shadows/white balance/denoise 편집.

### iPhone Camera Control / Exposure / Depth

**공식 확인 조작**
- Camera Control에서 Exposure와 Depth 조정 가능.
- 초점/노출 잠금 기능을 활용할 수 있다.

**실전 팁**
- 인물: Depth를 너무 낮은 f값처럼 과하게 주지 말고 머리카락/안경 경계를 확인한다.
- 달/네온: 노출을 낮춰 하이라이트를 먼저 살린다.

### Pixel Night Sight 공식 조작

**공식 경로/조작**
- Photo mode에서 Night Sight 사용.
- 저조도에서는 자동으로 동작할 수 있다.
- slider를 Max → Off 범위로 조정할 수 있다.
- More light = None으로 Night Sight를 끌 수 있다.

**실전 팁**
- 음식/인물은 Max보다 짧게, 정적인 풍경은 더 길게.
- 움직이는 사람에게는 몇 초간 정지를 요청한다.

### Pixel Pro controls / Macro / 50MP

**공식 확인 기능**
- Macro Focus.
- Pro controls.
- High-Res up to 50MP on supported devices.

**실전 팁**
- 음식 질감: Macro Focus 또는 2x/Portrait 비교.
- 여행 디테일: 50MP/High-Res는 큰 크롭이 필요한 장면에만.
- WB가 틀어지는 실내 음식은 Pro controls로 white balance를 먼저 확인.
