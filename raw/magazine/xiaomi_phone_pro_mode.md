# Xiaomi Support — What is phone camera Pro mode

## 바로 쓰는 팁 카드

### Tip 1. Pro mode의 핵심: WB / F / S / ISO / EV / LENS / Metering

**상황**
- 자동 모드에서 색/노출/초점이 원하는 결과와 다를 때.

**촬영/작업 순서**
- Camera → Pro mode 진입.
- 장면에 따라 WB, Focus, Shutter, ISO, EV, Lens, Metering 순으로 조정.

**추천 시작값 / 조작값**
- WB: 조명 환경(incandescent/daylight/cloudy)에 맞춰 우선 선택.
- Shutter: 정지 피사체는 느리게, 움직임 피사체는 빠르게.
- ISO: 저조도에서는 증가, 밝은 곳은 낮게.
- EV: 저조도는 +, 과다광은 -.

**주의점**
- 항목을 한 번에 크게 바꾸지 말고 한 스텝씩 조정해야 원인 파악이 쉽다.

**확실성**
- source-supported. Xiaomi 지원 문서의 항목 정의를 그대로 반영.

### Tip 2. 저조도 인물/실내: ISO를 무작정 올리지 않기

**상황**
- 실내/야간에서 사진이 어둡거나 흔들릴 때.

**촬영/작업 순서**
- 셔터, ISO, EV를 함께 확인해 노출 균형을 맞춘다.
- Focus peaking / exposure verification 기능을 켠 뒤 미세 조정.

**추천 시작값 / 조작값**
- ISO는 필요한 만큼만 올리고, EV와 셔터를 같이 조절.
- 노출 경고(under/over highlight)를 참고해 클리핑 최소화.

**보정 루틴**
- 하이라이트/섀도우 복구 후 색온도 미세 조정.

**주의점**
- ISO만 과도하게 올리면 노이즈가 빠르게 증가.

**확실성**
- Xiaomi 문서의 exposure verification/focus peaking 설명 기반.

### Tip 3. 렌즈와 측광을 의도적으로 고르기

**상황**
- 역광, 스팟 조명, 제품 촬영처럼 측광 기준점이 중요한 장면.

**촬영/작업 순서**
- LENS를 장면에 맞게 선택(ultra-wide / wide / tele).
- Metering 모드를 center / full / spot 중 장면에 맞게 선택.

**추천 시작값 / 조작값**
- 인물/제품 중심 노출: spot 또는 center 우선 테스트.
- 배경 포함 균형 노출: full frame metering 우선 테스트.

**주의점**
- 모델별로 사용 가능한 렌즈가 다르므로 장치 가용성을 먼저 확인.
