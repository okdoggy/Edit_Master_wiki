# Dimpy Bhalotia — 상황별 촬영/보정 팁

이 파일은 더 이상 단순 소스 메타데이터가 아니라, 실제 촬영/보정 때 바로 써먹는 레시피 형태로 정리한다.

**이 소스를 읽는 관점:** 수상작 분석용: 빛, 그림자, 결정적 순간, 그래픽 구도에 집중한다.

## 바로 쓰는 팁 카드

### Tip 1. 빛·그림자·실루엣이 강한 장면을 작품처럼 만들 때

**촬영/작업 순서**
- 컬러보다 밝기 구조를 본다. 강한 햇빛, 계단, 창문, 벽 그림자, 사람 실루엣을 찾는다.
- 노출은 밝은 부분 기준으로 낮춰 그림자를 깊게 만든다.
- 사람이 그래픽한 위치에 들어올 때까지 기다린다.

**추천 시작값 / 조작값**
- EV: -0.7~-1.3 시작
- Lens: 1x/2x
- Burst/Live: 사람 위치가 중요할 때

**보정 루틴**
- B&W 전환 후 Contrast +10~+30
- Highlights는 복구하고 Blacks를 내려 그래픽하게
- Grain은 소량만 추가

**주의할 점**
- 어두운 사진과 강한 사진은 다르다. 주 피사체 윤곽이 읽혀야 한다.
- Clarity 과다는 스마트폰 노이즈를 강조한다.

**확실성:** IPPAWARDS/스트리트 작가 사례 기반 실전 시작점.
**근거:** Dimpy Bhalotia, Brendan Ó Sé류의 수상/스트리트 모바일 사진에서 추출한 접근.

### Tip 2. 거리·여행·일상에서 순간을 놓치지 않고 찍을 때

**촬영/작업 순서**
- 카메라 앱을 미리 열고 1x 또는 2x를 기본값으로 정해 둔다.
- 빛이 좋은 곳을 먼저 찾고, 사람이 그 빛 안으로 들어올 때 기다린다.
- 움직임이 예측되면 Burst/Live Photo/짧은 영상으로 확보한 뒤 결정적 프레임을 고른다.
- 한 장보다 같은 장소의 와이드·중간·디테일 3컷을 묶어 스토리를 만든다.

**추천 시작값 / 조작값**
- Lens: 1x 환경컷, 2x/3x 인물·디테일
- EV: 강한 빛/그림자는 -0.3~-1.0
- Grid: 켜기
- Live/Burst: 움직임 많은 장면

**보정 루틴**
- 시리즈 전체의 WB/Contrast를 맞춘다.
- 하이라이트 기준으로 톤을 잡고 Shadows를 과하게 열지 않는다.
- 색이 산만하면 흑백 전환 후 밝기 구조를 본다.

**주의할 점**
- 줌으로 멀리서 훔치듯 찍는 대신 거리/각도/기다림으로 장면을 만든다.
- 모든 컷을 과보정해 시리즈 일관성을 깨지 않는다.

**확실성:** creator/editorial 사례 기반 실전 레시피.
**근거:** Sam Youkilis, Brendan Ó Sé, Benjamin Lowy 같은 모바일 다큐멘터리/스트리트 사례에서 추출.

### Tip 3. 무엇을 찍어도 사진이 산만하고 초점/밝기가 흔들릴 때

**촬영/작업 순서**
- 렌즈를 먼저 닦고, 카메라 설정에서 격자(Grid)와 수평(Level)을 켠다.
- 주 피사체를 화면 교차점 또는 중앙보다 약간 옆에 둔다.
- 피사체를 탭해서 초점과 노출 기준을 정하고, iPhone은 길게 눌러 AE/AF Lock을 건다.
- 하이라이트가 날아가면 화면 노출 슬라이더를 살짝 내려 밝은 부분 디테일을 살린다.

**추천 시작값 / 조작값**
- EV/노출 보정: 밝은 하늘·흰 벽·네온이 있으면 -0.3~-1.0부터 테스트
- 타이머: 삼각대/거치 시 2~3초
- 줌: 가능하면 0.5x/1x/2x/3x/5x 같은 광학 배율만 사용

**보정 루틴**
- 수평/크롭을 먼저 맞춘 뒤 노출·색을 조정한다.
- 밝은 영역 복구가 필요하면 Highlights를 낮추고 Shadows는 과하게 올리지 않는다.

**주의할 점**
- 디지털 줌으로 먼저 당기지 말고, 고해상도 촬영 후 크롭한다.
- 초점 확인 없이 연속 촬영하지 않는다.

**확실성:** source-supported 일반 원칙 + 실전 시작점.
**근거:** Apple의 AE/AF/Night mode 흐름, Samsung Pro mode, TIME/PetaPixel/Digital Camera World류의 기본 팁을 통합.

### Tip 4. 사진 전체 필터가 아니라 특정 색만 고치고 싶을 때

**촬영/작업 순서**
- 현장에서 색이 중요한 피사체를 의식하고, 같은 장소에서 밝은 컷/어두운 컷을 하나씩 확보한다.

**추천 시작값 / 조작값**
- Color → Mix
- 피부: Orange Saturation -5~+5, Luminance +0~+10부터 조심스럽게
- 하늘: Blue Saturation -10~+15, Luminance -5~-20 시작
- 잔디: Green/Yellow Saturation -10~-30로 산만함 줄이기

**보정 루틴**
- Targeted Adjustment Tool로 이미지 위 색을 직접 찍고 조정한다.
- Hue는 크게 움직이면 색이 가짜처럼 보이므로 작게 움직인다.
- Before/After를 눌러 과보정 여부를 확인한다.

**주의할 점**
- 피부색과 배경 주황/노랑이 함께 바뀔 수 있으므로 마스크와 병행한다.
- 채도만 올리면 저렴한 필터 느낌이 난다.

**확실성:** Adobe 공식 Color Mix 절차 + 실전 시작점.
**근거:** Adobe Learn은 Color Mix에서 색상별 Hue/Saturation/Luminance와 Targeted Adjustment Tool 사용을 설명한다.
