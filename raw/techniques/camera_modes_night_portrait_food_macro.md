# 야간·인물·음식·매크로 모드별 촬영 가이드

## Tip 1. 야간 모드 — “밝게”가 아니라 “흔들림 없이 필요한 만큼”

**상황**
- 밤거리, 실내 저조도, 네온, 야경, 조명이 약한 인물.

**촬영/작업 순서**
- iPhone: Night mode가 자동으로 켜지면 노출 시간을 확인하고, 정적 장면은 길게, 움직이는 사람은 짧게 간다.
- Pixel: Night Sight가 뜨면 사용하고, 아주 어두우면 피사체 주변을 탭해 초점 보조.
- Galaxy: More → Night 또는 Pro/Expert RAW 장노출을 선택한다.
- 공통: 벽/난간/삼각대에 고정하고 2~3초 타이머를 쓴다.

**추천 시작값 / 조작값**
- iPhone 정적 야경: Night mode Max 또는 긴 옵션 + 십자선 정렬.
- Pixel 인물 야간: 사진가가 피사체 4ft 이내, 피사체는 배경에서 6ft 이상 떨어지면 bokeh가 좋아짐.
- Galaxy 도시 야경: Pro mode 시작점 shutter 8s, ISO 50, WB 4300K, focus infinity.

**보정 루틴**
- Highlights -30~-70, Shadows +10~+35.
- Noise Reduction은 먼저 Color Noise를 확인하고 과하게 밀지 않는다.
- 네온은 Select Subject/Background/Sky로 부분 보정.

**주의할 점**
- 움직이는 사람과 장노출은 잘 맞지 않는다. 사람은 짧게, 풍경은 길게.
- 달은 야간 모드보다 Pro/Expert RAW 저ISO/빠른 셔터가 유리한 경우가 많다.

## Tip 2. 인물 모드 — 배경을 흐리는 것보다 거리 설계가 먼저

**상황**
- SNS 프로필, 카페 인물, 야간 조명 배경 bokeh, 여행지 인물.

**촬영/작업 순서**
- 피사체와 배경 사이를 벌린다. 배경 조명은 멀수록 예쁜 원형 bokeh가 된다.
- iPhone: Portrait mode → Portrait Lighting → 촬영 후 Depth Control.
- Pixel: Portrait mode, 어두우면 Night Sight in Portrait가 자동 관여할 수 있다.
- Galaxy: Portrait에서 Blur / Studio / High-key mono / Low-key mono / Backdrop / Color point 등을 선택한다.

**추천 시작값 / 조작값**
- iPhone Photography School식 기본 거리: 2ft 이상, 8ft 이하.
- Pixel 공식 팁: 사진가는 피사체 4ft 이내, 피사체는 배경에서 6ft 이상.
- 렌즈: 2x/3x는 얼굴 왜곡이 적고, 1x는 좁은 실내에 유리.

**보정 루틴**
- Subject mask: Exposure +0.1~+0.4, Shadows +5~+20.
- Background: Saturation -5~-20, Texture/Clarity -5~-15.
- 피부: Orange Saturation/Luminance는 ±5~10 내에서 작게.

**주의할 점**
- 머리카락, 안경, 손가락 경계가 깨지면 Depth/Blur를 줄인다.
- 0.5x 인물은 가장자리 얼굴 왜곡이 크다.

## Tip 3. 음식 모드 — “맛있게” 보이지만 과포화는 줄이기

**상황**
- 식당 음식, 카페 디저트, 홈쿡, 배달 메뉴 사진.

**촬영/작업 순서**
- Galaxy: Camera → More 또는 Mode → Food/Food Mode.
- 접시가 강조 영역 안에 들어오도록 highlighted area를 드래그한다.
- 필요하면 영역 모서리를 드래그해 크기를 조절한다.
- 지원 기기에서는 색 조절과 radial blur/blur effect 아이콘을 확인한다.
- iPhone/Pixel에는 전용 Food mode가 없으면 1x/2x + Portrait 또는 일반 Photo + 후편집으로 대응한다.

**추천 시작값 / 조작값**
- 밝은 식당/창가: 2x + 45도 각도.
- 플랫레이: 1x 또는 0.5x, Grid/Level 켜기, 손/컵/식기 여백 확보.
- 어두운 식당: 플래시 금지, 창가/테이블 조명/촛불을 활용, EV -0.3.
- Galaxy Food mode가 너무 노랗거나 과포화면 일반 Photo/Portrait로도 한 장 찍어 비교.

**보정 루틴**
- WB 먼저: 흰 접시가 노랗거나 파랗지 않게.
- Vibrance +5~+12, Saturation +0~+8.
- Orange/Yellow Saturation +5~+15, Luminance +0~+10.
- Texture +10~+20, Clarity +5~+10.
- 가장자리 radial blur가 과하면 끄거나 줄인다.

**주의할 점**
- 플래시는 음식 표면을 기름지고 납작하게 만든다.
- Food mode의 색 강조는 메뉴판/기록용에는 과할 수 있다.

## Tip 4. 매크로/근접 모드 — 가까이보다 “초점 거리”가 중요

**상황**
- 꽃, 디저트 질감, 커피 크레마, 제품 디테일, 작은 장식.

**촬영/작업 순서**
- iPhone: 지원 모델에서 최대 2cm 접근 시 ultrawide macro로 전환 가능. 흐리면 물러나거나 .5x를 탭한다.
- Pixel: Macro Focus는 피사체에 가까이 가면 자동으로 관여한다. 빛이 부족하면 먼저 밝은 곳으로 옮긴다.
- Galaxy: S23/Fold5 계열은 0.6x, Flip5 계열은 0.5x 접근을 시도하고, 30cm 이내에서 흐리면 약 2.5cm 더 물러난다.

**추천 시작값 / 조작값**
- 음식 디테일: 2x로 한 발 물러나 압축감을 주는 컷도 함께 촬영.
- 꽃/제품: 배경과 30cm 이상 거리 확보.
- 바람/손떨림: Burst 또는 짧은 영상으로 여러 프레임 확보.

**보정 루틴**
- Texture +10~+25, Clarity +5~+15.
- 배경은 Saturation/Clarity를 낮추고, 피사체만 마스크로 살린다.

**주의할 점**
- 너무 가까이 가면 자동 렌즈 전환 때문에 구도가 튈 수 있다.
- 주변부 흐림이 싫으면 카메라를 피사체와 평행하게 유지한다.

## Tip 5. Pro / Expert RAW — 자동 모드가 틀리는 장면에서만 꺼내기

**상황**
- 달, 야경, 공연, 역광, 색온도가 어려운 실내, 보정할 작품 컷.

**촬영/작업 순서**
- 하이라이트가 날아가지 않게 어둡게 찍는다.
- ISO는 가능한 낮게 유지하고, 셔터는 피사체 움직임에 맞춘다.
- WB는 Auto로 찍어도 RAW에서 조정 가능하지만, 시리즈는 Kelvin을 고정하면 색 일관성이 좋아진다.

**추천 시작값 / 조작값**
- 달: ISO 50~100, 1/125~1/500s, MF 무한대.
- 야간 도시: ISO 50~200, 2~8s, 삼각대.
- 실내 인물: ISO는 자동보다 낮게, 셔터 1/60~1/125s 확보.

**주의할 점**
- RAW는 용량과 편집 시간이 든다. 모든 스냅에 켜지 말고 “보정할 컷”만 쓴다.

## 모드별 공식 검증값 추가

### Samsung Food mode 세부 조작

**공식 경로**
- Camera → MORE → FOOD.

**촬영 절차**
- 음식이 highlighted area 안에 들어오도록 프레임을 잡는다.
- Thermometer 아이콘으로 색온도/색감을 조정한다.
- Blur effect로 음식 주변이나 배경을 흐린다.

**실전 팁**
- 흰 접시가 노랗게 뜨면 Thermometer를 차갑게 조정한다.
- 여러 접시가 있으면 Blur effect를 약하게 하거나 일반 Photo도 같이 찍는다.

### Samsung Night mode 세부 원리

**공식 경로**
- Camera → More → Night → Capture.

**공식 확인 포인트**
- Samsung Night Mode는 AI multi-frame processing으로 여러 이미지를 합성한다.
- Samsung night-mode guide는 30 images를 하나로 결합하는 설명을 제공한다.
- 안정적인 자세 또는 삼각대가 결과를 크게 좌우한다.

**실전 팁**
- 손持ち: 짧은 Night mode, 사람은 멈추게 하기.
- 삼각대/고정: 더 긴 Night mode 또는 Pro mode 장노출.
- 달/네온처럼 매우 밝은 피사체는 Night mode보다 수동 노출이 유리할 수 있다.

### Samsung Pro mode / Expert RAW 범위

**공식 확인 조작**
- ISO, shutter speed, exposure, color tone, manual focus, white balance.

**공식 예시 범위**
- ISO 50–800.
- Shutter 1/24000–10s.
- White balance 2300K–10000K.
- Manual focus: 가까운 거리부터 far away.

**Expert RAW 예시**
- 별도 다운로드 앱/기능.
- 더 세밀한 제어와 RAW 기반 편집.
- Samsung S25 자료 예시: ISO 800, 1/125s, AF-C, highlights/shadows/white balance/denoise 편집.

### iPhone Camera Control / Exposure / Depth

**공식 확인 조작**
- Camera Control에서 Exposure와 Depth 조정 가능.
- 초점/노출 잠금 기능을 활용할 수 있다.

**실전 팁**
- 인물: Depth를 너무 낮은 f값처럼 과하게 주지 말고 머리카락/안경 경계를 확인한다.
- 달/네온: 노출을 낮춰 하이라이트를 먼저 살린다.

### Pixel Night Sight 공식 조작

**공식 경로/조작**
- Photo mode에서 Night Sight 사용.
- 저조도에서는 자동으로 동작할 수 있다.
- slider를 Max → Off 범위로 조정할 수 있다.
- More light = None으로 Night Sight를 끌 수 있다.

**실전 팁**
- 음식/인물은 Max보다 짧게, 정적인 풍경은 더 길게.
- 움직이는 사람에게는 몇 초간 정지를 요청한다.

### Pixel Pro controls / Macro / 50MP

**공식 확인 기능**
- Macro Focus.
- Pro controls.
- High-Res up to 50MP on supported devices.

**실전 팁**
- 음식 질감: Macro Focus 또는 2x/Portrait 비교.
- 여행 디테일: 50MP/High-Res는 큰 크롭이 필요한 장면에만.
- WB가 틀어지는 실내 음식은 Pro controls로 white balance를 먼저 확인.
